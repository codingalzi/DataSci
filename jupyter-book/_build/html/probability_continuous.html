13.1. 연속 확률 변수#

12장에서 살펴 본 이산 확률 변수는 \(x_1\), \(x_2\), \(x_3\) … 등의 형식으로 하나, 둘, 셋 하며 셀 수 있는 이산형 값discrete value을 가리킨다.

반면에 연속 확률 변수continuous random variable는 일정 구간에 속한 임의의 실수를 가리킨다. 여기서 구간은 0과 1사이 등 구간의 크기가 유한할 수도 있지만 0보다 큰 모든 실수, 아니면 그냥 모든 실수 처럼 구간의 크기가 무한일 수도 있다.

실수와 부동소수점

프로그래밍에서 실수는 부동소수점으로 표현된다. 부동소수점은 엄밀히 말해 유한소수, 그것도 제한된 수준의 정확도만 보장한다. 하지만 여기서는 소수점 이하 몇째 자리까지 정도로 표현된 소수만을 대상으로 하기에 그런 한계는 고려하지 않는다.

연속 확률 변수가 따르는 확률 분포를 연속 확률 분포continuous probability distribution라 한다. 이산 확률 변수에 대한 확률 분포와는 달리 연속 확률 분포는 확률 변수가 특정 값을 취할 확률을 계산하지 않는다. 대신 확률 변수가 취하는 값이 특정 구간에 속할 확률을 계산한다.

이유는 연속 확률 변수 \(X\)가 특정 부동소수점 \(x\)를 취할 확률은 0이기 때문이다

\[ P(X=x) = 0 \]

예를 들어 0과 1사이의 부동소수점 중에서 무작위로 하나의 값을 취했을 때 그 값이 정확히 0.5일 확률은 0이다. 반면에 무작위로 선택한 값이 0.49에서 0.51 사이일 확률은 0.02, 즉 2%다. 구간 전체의 크기가 1인데 0.49에서 0.51 사이 구간의 크기는 0.02이기 때문이다.

이처럼 연속 확률 변수는 특정 값이 아닌 특정 구간의 값을 취할 확률을 계산한다.

\[ P(a \le X \le b) \]

누적 분포 함수

연속 확률 변수 \(X\)가 \(a\)보다 같거나 작은 값을 취할 확률을 반환하는 함수를 \(F(a)\)라 표기하고 이산 확률 분포의 경우처럼 누적 분포 함수cumulative distribution function(CDF)라 부른다.

\[ F(a) = P(X \le a) \]

누적 분포 함수를 이용하여 연속 확률 변수 \(X\)가 \(a\)에서 \(b\) 사이의 값을 취할 확률을 다음과 같이 계산할 수 있다. 단, \(a \le b\)를 가정한다.

\[ P(a \le X \le b) = F(b) - F(a) \]

연속 확률 변수 \(X\)가 \(x_0\)에서 \(x_1\) 사이의 값을 취할 때 \(X\)의 누적 분포 함수\(F\)는 다음 세 가지 성질을 만족한다. 단, \(a, b \in [x_0, x_1]\)과 \(a \le b\)를 가정한다.

\((1)\) \(F(a) \le F(b)\)
\((2)\) \(0 \le F(a) \le 1\)
\((3)\) \(\displaystyle \lim_{a \to x_0} F(a) = 0\) 와 \(\displaystyle\lim_{a \to x_1} F(a) = 1\)

위 식에서 경우에 따라 \(x_0\)는 음의 무한대를, \(x_1\)은 양의 무한대를 가리킬 수 있음에 주의한다. 예를 들어 다음이 성립한다.

확률 밀도 함수

앞서 설명한 대로 연속 확률 변수가 특정 값을 취할 확률은 0이기에 확률 질량 함수는 정의하지 않는다. 반면에 연속 확률 변수가 특정 구간 내에서 값을 취할 확률은 계산할 수 있기에 확률 변수가 특정 값 주변에 위치할 확률은 계산할 수 있다.

예를 들어 연속 확률 변수 \(X\)가 구간 \([x, x+h]\)에서 값을 취할 확률은 다음과 같다. \(h\)는 임의의 양의 실수다.

\[ \frac{F(x + h) - F(x)}{h} \]

즉, 위 식은 연속 확률 변수 \(X\)가 값 \(x\) 주변에 위치할 상대적 가능성을 계산한다. 그리고 만약 함수 \(F\)가 \(x\)에서 미분가능하다면 위 값은 \(h\)가 충분히 작을 때 특정 값으로 수렴한다. 즉, 다음을 만족시키는 값 \(f(x)\)가 존재한다.

\[ f(x) = \lim_{h \to 0}\frac{F(x + h) - F(x)}{h} \]

만약 함수 \(F\)가 구간 \([x_0, x_1]\)에서 미분가능하다면 임의의 \(x \in [x_0, x_1]\)에 대해 다음이 성립한다.

\[\begin{split} f(x) = \frac{dF(x)}{dx} = F'(x)\\[.5ex] \end{split}\]

특히 다음 관계가 성립하기에 함수 \(f\)를 연속 확률 변수 \(X\)의 확률 밀도 함수probability density function(PDF)라 부른다.

\[ F(a) = P(X \le a) = \int_{x_0}^{a} f(x)\, dx \]

13.1.1. 데이터 전처리: 표준화#

특성에 따라 사용되는 값들의 크기 정도scale가 다르다. 어떤 특성은 0과 1사이의 값을, 다른 특성은 100단위의 값을 포함하기도 한다. 그런데 머신러닝 모델은 기본적으로 모든 특성이 동일한 크기 정도의 값으로 구성될 때 보다 잘 훈련된다. 이런 이유로 여기서는 평균은 0, 표준편차는 1이 되도록 변환하는 표준화standardization를 적용해서 훈련셋과 테스트셋을 전처리한다.

표준화는 다음 식으로 계산된다.

\[ \frac{x - \mu}{\sigma} \]

\(x\)는 샘플의 특성값을, \(\mu\)와 \(\sigma\)는 특성별 평균값과 표준편차를 가리킨다. 넘파이 어레이를 이용하면 전체 훈련셋에 대해 다음처럼 한 번에 표준화를 진행할 수 있다.

# 훈련셋의 특성별 평균값/표준편차
mean = train_data.mean(axis=0)
std = train_data.std(axis=0)

# 훈련셋 표준화
train_data -= mean
train_data /= std
